# Sai Vishal Matcha - Portfolio

A clean, simple, and functional portfolio website showcasing professional projects and experience.

## Features

- ✨ Responsive design (mobile, tablet, desktop)
- 🎨 Dark theme with blue accents
- ⚡ Pure HTML/CSS/JavaScript (no build tools)
- 📱 Mobile-friendly navigation
- 🔗 Social media and contact links
- 🚀 Deployed on GitHub Pages

## Sections

- **About** - Professional background and current focus
- **Skills** - Technical expertise across different domains
- **Experience** - Career history and achievements
- **Projects** - Featured work and portfolio items
- **Contact** - Ways to reach out
